# Encode Fellowship — Application
In this repository I am going to document the process of Encode Fellowship application preparation and submission.

### Overview of the application requirements
Track:
Non-profit Institution

ARIA opportunity space:
Collective Flourishing

The deliverables (the ones that require preparation):
1. Application form website (all following entries 2500 chars max)
    1.1. Tell us about yourself. What’s your most ambitious     goal and how does the Encode fellowship help you get there?
    1.2. What technical project/achievement are you most proud of and why?
    1.3. Do you have any experience working on cross-disciplinary teams?
2. Demo of how I work.
    Help us understand how you work - please demo a technical project you’ve worked on in less than 3 minutes.
    Please provide a URL - youtube, google drive.
3. Application in the form of Google document
    (https://docs.google.com/document/d/1YIqjkf9p3GZy1t6IIdlGDVcYFmnSb_1aYGL0Q43HU2s/edit?pli=1&tab=t.0)
